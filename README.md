# Brander Cloud — Starter project
Proyecto inicial **Brander Cloud**. Stack: Next.js + React + Tailwind CSS.

## Objetivo
Plantilla base para iniciar proyectos web para empresas en Colonia del Sacramento. Incluye ejemplo de página y meta-descripciones optimizadas.

## Metadescripciones sugeridas (usar una en la etiqueta `meta name="description"`):
- "Soluciones web en Colonia del Sacramento"
- "Creamos la web que tu negocio necesita en Colonia del Sacramento, Uruguay"

## Cómo usar
1. Instalar dependencias:
   ```bash
   npm install
   # o
   yarn
   ```
2. Ejecutar en modo desarrollo:
   ```bash
   npm run dev
   ```
3. Construir para producción:
   ```bash
   npm run build
   npm run start
   ```

## Estructura relevante
- `pages/` — Rutas Next.js (incluye `index.js` y `_app.js`)
- `components/` — Componentes React reutilizables (`Header`, `Hero`, `Footer`)
- `styles/globals.css` — Tailwind base
- `public/` — Archivos públicos (favicon)
